# Metzler Homepage

Static homepage redesign for [edelstahl-tuerklingel.de](https://edelstahl-tuerklingel.de) — premium UX/UI/CRO rebuild of the Metzler GmbH landing page.

## Stack

- Plain **HTML + CSS** — no preprocessor, no framework, no build step
- ~15 lines of inline vanilla JavaScript for the mobile nav toggle

## Run locally

Open `index.html` directly in any modern browser. No server required.
